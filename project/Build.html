Building Plex Media Center

Windows

Repository maintenance

Ensure your PMC repository is up to date, by running the following commands from the root of the repo:

  • git submodule init
  • git submodule update

Building PMC

  • Open a Visual Studio command prompt and run project\BuildDependencies\ong>DownloadBuildDeps.bat. Let it complete.
    • This generally only needs to be done once.
  • Using Visual C++ 2010, open the solution file located at project\VS2010Express\Plex for Windows.sln.
  • Cmpile the Release (Direct X) or Debug (DirectX) projects

Building a setup package

  • Open a Visual Studio command prompt.
  • Run project\Win32BuildSetup\BuildSetup.bat

Building ffmpeg for PMC:

  • Open an msys command prompt
  • ./configure-windows.sh
  • ./make

WhenWhen a new build of ffmpeg is generated, the following files must be copied into the system\players\dvdplayer directory.

  • avcodec-53.dll
  • avformat-53.dll
  • avutil-51.dll
  • postproc-51.dll
  • swscale-2.dll

In addition, the libavutil\avconfig.h header must be copied into the lib\ffmpeg\libavutil directory.
